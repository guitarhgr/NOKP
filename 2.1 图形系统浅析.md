## 计算机图形系统的组成

- 输入设备

  ```
  用于输入数据的装置，从原始的穿孔纸到数据手套，数据衣
  ```

  

- 显示设备

  ```
  主要指各类显示器，用于显示输出图形信息,例如：CRT显示器，液晶显示器，3D显示
  ```

  

- 输出设备

  ```
  主要指硬拷贝设别，用于将图形或非图形信息以打印或印刷的形式长期保存，例如：打印机，绘图仪。
  ```
